새 폴더 생성하면 cmd 창에 해당 폴더 열고 아래 프로그램 설치

1.해당 폴더 여는 법
cd (폴더 경로 복사,"" 제거하고 enter)

2. npm install express
(단축키 등 )

3. npm install ws
(웹소켓인데 클라이언트와 서버의 연결을 빠르게 해준다.. 네요)

4. npm install mongoose
(몽고디비 연결용)

5. npm install cors 

이외 프로그램 설치 다 하면
node (서버파일 이름).js 
하면 서버 구동 시작!


폴더에 프로그램 한 번 설치하면 이후에는 node 파일이름.js 이렇게 해서 서버연결만 하면 됨!
이외에도 코드 만들다가 더 설치한 모듈들이 잇는데 사실상 코드 맨 윗부붑ㄴ에 require 이런식으로 된 것들 다 모듈이니까 새로 까시면 됩니다!
